Algorithm Information

Bubble Sort

Compares adjacent elements and swaps them if they're in wrong order.

Time Complexity: O(n²)

Selection Sort

Finds the minimum element and places it at the beginning.

Time Complexity: O(n²)

Insertion Sort

Builds the sorted array one item at a time.

Time Complexity: O(n²)
